2. MÔ TẢ DỰ ÁN:
   - Thể loại: Turn-Based RPG (Game nhập vai theo lượt)
   - Công nghệ: Java / Greenfoot Framework
   - Mục tiêu: Phát triển hệ thống chiến đấu theo lượt với cơ chế 
     xử lý AI Intent, hệ thống Buff ngẫu nhiên và quản lý thực thể.

3. HƯỚNG DẪN CHẠY DỰ ÁN:
   - Bước 1: Cài đặt Greenfoot (phiên bản mới nhất).
   - Bước 2: Mở ứng dụng Greenfoot.
   - Bước 3: Chọn 'Scenario' -> 'Open' -> Tìm đến thư mục dự án này.
   - Bước 4: Nhấn nút 'Run' để khởi chạy trò chơi.

4. HƯỚNG DẪN ĐIỀU KHIỂN:
   - Sử dụng chuột trái để tương tác.
   - Click chọn quái vật mục tiêu trước khi thực hiện hành động.
   - Nhấn các nút chức năng (TẤN CÔNG, KỸ NĂNG, BÌNH MÁU) 
     để điều khiển nhân vật chính (Hero).

5. LƯU Ý VỀ KIẾN TRÚC:
   - Hệ thống được thiết kế dựa trên Mediator Pattern (MyWorld) 
     và State Pattern (Hero logic) nhằm tối ưu hóa khả năng 
     bảo trì và mở rộng code.
